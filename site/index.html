ようこそ

本ポータルは、主要ドキュメントとテストレポートへ素早く辿るための入口で御座る。

  • 初めての方は セットアップ
  • チーム規約は 開発ルール
  • E2Eの結果は 最新レポート

クイックコマンド

# 依存関係
npm ci

# Prisma (ローカル)
npx prisma generate
npx prisma db push

# Dev
npm run dev

# E2E（Stripeモック例）
export E2E_STRIPE_MOCK=1
npx playwright test
