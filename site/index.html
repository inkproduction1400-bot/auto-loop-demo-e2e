ようこそ

本ポータルはプロジェクトの“入口”であり、主要ドキュメントとテストレポートへ素早く辿れることを目的とするで御座る。

  • 初めての方は セットアップ へ
  • チーム規約は 開発ルール を参照
  • E2Eの結果は 最新レポート に常時導線

クイックコマンド

# 依存関係
npm ci

# Prisma (ローカル)
npx prisma generate
npx prisma db push

# Dev
npm run dev

# E2E（Stripeモック例）
export E2E_STRIPE_MOCK=1
npx playwright test
